# GEPT 詞彙互動測驗平台 - 設計理念

## 設計風格：現代教育科技 (Modern EdTech)

### 核心原則
1. **清晰的信息層級**：通過色彩、大小和間距引導用戶注意力
2. **即時反饋與激勵**：每個答案選擇都提供視覺化的反饋，鼓勵學習
3. **流暢的交互體驗**：平滑的過渡和動畫增強用戶參與感
4. **可訪問性優先**：確保所有用戶都能輕鬆使用

### 色彩哲學
- **主色調**：藍色系 (#3B82F6) - 代表信任、學習和專業性
- **成功反饋**：綠色 (#10B981) - 正確答案的即時視覺確認
- **錯誤反饋**：紅色 (#EF4444) - 錯誤答案的清晰提示
- **背景**：白色和淺灰色 - 保持清爽、專注的學習環境
- **文字**：深灰色 (#1F2937) - 高對比度，易於閱讀

### 版面設計
- **居中對稱佈局**：營造專注感，減少視覺干擾
- **卡片式設計**：每個問題獨立呈現，清晰明確
- **充足的空白**：提高可讀性和視覺舒適度

### 特徵元素
1. **進度指示器**：顯示當前問題在整個測驗中的位置
2. **即時反饋卡片**：答案選擇後立即顯示正確/錯誤狀態
3. **詞彙解釋面板**：提供中文解釋和例句

### 交互設計
- **平滑過渡**：頁面和卡片之間的淡入淡出效果
- **懸停效果**：選項卡在懸停時改變顏色和陰影
- **點擊反饋**：選項被選中時的視覺確認
- **進度動畫**：進度條的平滑更新

### 字體系統
- **標題字體**：使用較大的字體和粗體突出問題
- **正文字體**：清晰易讀的字體用於選項和解釋
- **字體大小層級**：標題 > 正文 > 輔助文本

## 用戶旅程
1. **歡迎頁面**：簡潔介紹和開始按鈕
2. **測驗頁面**：逐題呈現，每題完成後立即反饋
3. **結果頁面**：總結成績和表現分析
4. **重新開始**：允許用戶再次進行測驗

## 實現細節
- 使用 Tailwind CSS 進行樣式管理
- 使用 shadcn/ui 組件保持設計一致性
- 使用 Framer Motion 實現流暢的動畫效果
- 響應式設計，支持各種屏幕尺寸
